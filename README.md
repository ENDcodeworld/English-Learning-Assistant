## 🚀 快速开始

```bash
git clone https://github.com/ENDcodeworld/English-Learning-Assistant.git
cd English-Learning-Assistant
pip install -r requirements.txt
python main.py
```

---

## 📦 打包APK

```bash
pip install buildozer
buildozer init
buildozer android debug
```
